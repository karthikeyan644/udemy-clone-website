## Udemy Clone Website

This project is a basic **Udemy clone website** built using **HTML** and **CSS**. It is a front-end project created to practice and strengthen fundamental web development skills.

The website replicates the layout and design of an online learning platform, featuring a structured homepage, navigation bar, and course listing sections. **HTML** is used to create the structure and content of the website, while **CSS** is used for styling, layout, and visual design.

### Features

* Simple and clean user interface
* Homepage layout similar to Udemy
* Course cards and sections
* Navigation bar
* Styled using basic CSS

### Technologies Used

* HTML5
* CSS3

### Purpose

The main purpose of this project is to gain hands-on experience with front-end development concepts such as page structure, styling, and layout design. This project serves as a foundation for future enhancements like adding JavaScript functionality, responsiveness, and backend integration.
